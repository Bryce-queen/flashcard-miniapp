# 缺陷修复 2025-06-13

## 修复列表

### 🔴 #1 drawWideSegments 缺少 fillStyle（严重）
- 问题：星空模板调用 drawWideSegments 时未设置 ctx.fillStyle，文字颜色来自上一次渲染的残留值，在暗底上可能不可见
- 修复：drawWideSegments 签名增加 textColor 参数，调处传入 tpl.text；方法内部在首次循环前 set ctx.fillStyle
- 文件：index.js

### 🟡 #2 预览卡不渲染 Markdown（中等）
- 问题：预览区 {{inputText}} 原样输出 **粗体** 和 # 标题标记，与 Canvas 渲染效果割裂
- 修复：新增 stripMarkdown(text) → 剥离 ** / # / ## / ### 标记；inputText 变更时同步计算 previewText；WXML 预览绑定 previewText
- 顺带：预览签注从 "闪卡 FlashCard" 精简为 "闪卡"，与 Canvas 一致
- 文件：index.js + index.wxml

### 🟡 #3 历史缩略图用临时路径（中等）
- 问题：wx.canvasToTempFilePath 返回的临时路径在用户退出小程序后失效，历史缩略图全裂
- 修复：导出 success 回调中用 FileSystemManager.saveFileSync 将临时文件持久化到 wx.env.USER_DATA_PATH/flashcard_{timestamp}.png；持久化失败时回退临时路径
- 文件：index.js

### 🟢 #4 parseBoldSegments 不处理未闭合 **（低）
- 问题：`**hello` 没有闭合 → "hello" 被错误地当作粗体片段
- 修复：内层循环后检查是否真的遇到了闭合 **，若未闭合则退回为普通文本（加回 ** 前缀）
- 文件：index.js
