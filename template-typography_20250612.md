# 模板深度重写 — 6 套独立排版语言

**时间**: 2025-06-12 16:30  
**决策**: 用户让 AI 选方向 → 选 A（模板深度），符合花叔"反 AI slop"最高优先级

---

## 改了什么

### 问题
6 套模板仅换配色。花叔设计哲学核心：「纯换色的模板就是 AI slop」。

### 方案
每个模板有自己的 Canvas 渲染指令，通过 `draw` 配置对象驱动。

### 六套排版语言

| 模板 | 关键差异化 |
|------|-----------|
| **暖白** | 基准：左竖线 + 上下发丝线 + 规则线签注。不改。 |
| **瑞士** | 1/3、2/3 位置两条极淡水平参考线（`alpha=0.18`，`lineWidth=0.5`）→ 网格秩序感 |
| **暗调** | 发丝线改 accent 暖橙 + 字号增大到 48px + 签注极简化 + 边距加宽 |
| **星空** | 居中排印 + 逐字宽间距（+5px） + 3-5 个伪随机星点（含十字星芒） + 签注居中 |
| **森林** | 无顶线 → 开放感 + 绿色竖线 + 字重 500（亲和） |
| **宣言** | 零装饰（无发丝线、无竖线） + 字号 56px + 字重 800 + 居中巨字 |

### 新增绘制方法

- `drawHairline()` — 发丝线，支持色值 override
- `drawLeftBar()` — 左装饰竖线
- `drawSwissGrid()` — 瑞士网格
- `drawStarDots()` — 星空小圆点 + 概率十字星芒（seed 来自文本 hash → 同文字同点位）
- `drawWideText()` — 逐字渲染 + 额外间距，支持居中对齐
- `drawColophon()` — 三种签注：`rule`（规则线）/ `centered`（居中）/ `minimal`（右下小字）
- `hashText()` + `seededRng()` — 伪随机工具
